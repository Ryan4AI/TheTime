## 三、属性增长规则

### 3.1 基础规则
1. **每次行为**：根据行为类型，按表中范围随机取值
2. **属性上限**：每个属性最高10000，达到后不再增长
3. **属性损耗**：某些行为可能同时消耗其他属性（如行侠可能损耗财富）
4. **难度系数**：
   - 乱世：战功、义行获取×1.5，财富获取×0.7
   - 盛世：财富获取×1.3，战功获取×0.8
   - 普通：正常获取

### 3.2 特殊规则
1. **颜值基本固定**：
   - 初始颜值由出身决定（3000-7000）
   - 可通过打扮小幅提升（每次+3~15）
   - 不会大幅下降（除非毁容事件）

2. **声望衰减**：
   - 长期不活跃，声望每月衰减1~3%
   - 英雄事迹可阻止衰减

3. **组合加成**：
   - 同时提升多个相关属性时，额外+10%
   - 如：治病同时提升医术+义行，两者各+10%

### 3.3 历史事件加成
| 事件类型 | 属性加成 | 说明 |
|---------|---------|------|
| 参与重大战役 | 战功×2, 声望×1.5 | 改变历史的战役 |
| 平定大乱 | 战功×2.5, 政绩×2, 声望×2 | 结束乱世 |
| 开创盛世 | 政绩×2, 声望×2, 财富×1.5 | 治理有方 |

| 医学突破 | 医术×2, 声望×1.5 | 重大医学成就 |

## 四、历史人物属性计算

### 4.1 计算方法
根据历史人物的**一生行为**，按照属性增长表累加计算：

**示例：华佗**
- 一生治病无数（假设治愈10000人）
  - 普通疾病：8000次 × (医术+10) = 医术+80000 → 上限10000
  - 疑难杂症：1500次 × (医术+40) = 医术+60000 → 已上限
  - 瘟疫救人：100次 × (医术+80) = 医术+8000 → 已上限
- 编写医书：2本 × (医术+100) = 医术+200
- **最终医术：10000**（达到上限）
- **声望计算**：
  - 治愈10000人 × (声望+5) = 声望+50000 → 上限10000
  - 编写医书 × (声望+30) = 声望+60
  - **最终声望：9500**
