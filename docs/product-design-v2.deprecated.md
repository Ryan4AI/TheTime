## 一、产品概述

| 项目 | 内容 |
|------|------|
| 产品名 | **穿越日记**（英文：TheTime） |
| 形态 | 微信小游戏（Canvas 2D） |
| 核心爽点 | **留名青史，进入历史名人榜，摆脱死神追杀** |
| 核心标签 | AI剧情 · 穿越 · 历史真实数据 · 名人榜 · 死神追杀 |

### 核心循环

```
选物品 → 随机穿越身份 → AI叙事+选项 → 积累属性 → 冲击榜单
   ↓                                                    ↓
  下一世 ← 遗言+墓志铭 ← 死神追杀/寿终正寝 ← 超越历史名人
```

## 二、属性体系

### 2.1 通用属性（4个）

| 属性 | 说明 | 范围 | 作用 |
|------|------|------|------|
| 声望 | 名气、人望 | 0-10000 | 所有榜单通用权重 |
| 财富 | 金钱、资产 | 0-10000 | 富商榜排序依据 |
| 学识 | 知识、学问 | 0-10000 | 文豪榜/全能榜权重 |
| 颜值 | 外貌、魅力 | 0-10000 | 颜值榜排序依据 |

### 2.2 专属属性（5个）

| 属性 | 说明 | 范围 | 对应榜单 |
|------|------|------|----------|
| 医术 | 医疗技能 | 0-10000 | 名医榜 |
| 战功 | 军事成就 | 0-10000 | 名将榜 |
| 文采 | 文学才华 | 0-10000 | 文豪榜 |
| 政绩 | 治国能力 | 0-10000 | 能臣榜 |
| 义行 | 侠义行为 | 0-10000 | 义士榜 |

### 2.3 属性增长规则

详见 `docs/attribute-rules.md`，核心要点：
- 8大类行为（医疗/军事/文学/政治/侠义/商业/学习/社交）各有对应属性增长
- 乱世加成：战功/义行 ×1.5，财富 ×0.7
- 盛世加成：财富 ×1.3，战功 ×0.8
- 历史事件加成：参与重大战役战功×2，平定大乱战功×2.5 等

## 三、历史名人榜系统

### 3.1 十大榜单

| 榜单 | 类型 | 排序公式 | 人物数 |
|------|------|----------|--------|
| 名医榜 | 专业 | 医术×0.7 + 声望×0.3 | 100 |
| 名将榜 | 专业 | 战功×0.7 + 声望×0.3 | 100 |
| 富商榜 | 专业 | 财富×1.0 | 100 |
| 文豪榜 | 专业 | 文采×0.7 + 学识×0.3 | 100 |
| 能臣榜 | 专业 | 政绩×0.7 + 声望×0.3 | 100 |
| 义士榜 | 专业 | 义行×0.7 + 声望×0.3 | 100 |
| 全能榜 | 专业 | 声望+财富+学识+颜值 | 100 |
| 长寿榜 | 趣味 | 寿命 | 30 |
| 旅行家榜 | 趣味 | 游历城市数 | 30 |
| 颜值榜 | 趣味 | 颜值×1.0 | 30 |

### 3.2 历史人物数据

- 总人物数：790人（7专业×100 + 3趣味×30）
- 数据文件：`data/leaderboards.json`
- 每个人物含：排名、姓名、朝代、各属性值、综合分、完整介绍（称号/传记/成就/名言/上榜理由）

### 3.3 玩家上榜规则

1. **进入门槛**：综合分 > 榜单最低人物的综合分
2. **排名计算**：按公式计算综合分，插入榜单对应位置
3. **多榜可上**：玩家可同时出现在多个榜单（如医术+文采双修可上名医榜+文豪榜）
4. **动态排名**：每次属性变化后重新计算排名
5. **超越提示**：当玩家综合分超过某历史人物时，叙事提示"你的医术已超越华佗！"

### 3.4 榜单交互设计

**榜单入口页：**
- 10个榜单 Tab 切换
- 每个榜单显示 Top 100
- 玩家当前位置高亮（如"你排第23位"）
- 点击历史人物 → 详情页（传记/成就/名言）

**回合反馈：**
- 每回合结束显示属性变化
- 显示榜单排名变化（如"名医榜：第45位→第38位 ↑7"）
- 超越历史人物时特殊提示

## 四、死神追杀机制

### 4.1 核心设定

> 死神（命运之轮）暗中追杀玩家。玩家通过积累属性、留名青史来获得"历史庇护"，摆脱死神。

### 4.2 危险度系统

| 危险等级 | 触发条件 | 死神行为 |
|---------|---------|---------|
| 安全 | 综合属性 > 5000 或 上榜 | 不出现 |
| 潜伏 | 综合属性 3000-5000 | 偶尔制造小意外 |
| 逼近 | 综合属性 1000-3000 | 频繁制造危机事件 |
| 紧迫 | 综合属性 < 1000 | 每回合都有生命危险 |
| 锁定 | 连续3回合无属性增长 | 必定触发致命事件 |

### 4.3 历史庇护

- 每上榜一次 → 获得 1 层历史庇护（最多 5 层）
- 每层庇护降低一个危险等级
- 5 层庇护 = 永久安全（成为历史传奇）

### 4.4 死神事件类型

| 类型 | 说明 | 应对 |
|------|------|------|
| 意外 | 坠马、溺水、中毒 | 属性检定（体质/反应） |
| 疾病 | 瘟疫、疑难杂症 | 医术检定 |
| 战乱 | 兵灾、匪患 | 战功/武力检定 |
| 阴谋 | 权贵陷害、背叛 | 声望/人脉检定 |
| 天灾 | 洪水、地震 | 运气检定 |

### 4.5 死亡与轮回

- 死亡后总结本世成就：获得属性、上榜情况、历史庇护层数
- 历史庇护层数带入下一世（保留部分）
- 跨世积累，最终目标是成为传奇

## 五、游戏流程（v2）

### 5.1 开局

```
入口页 → 选物品（30抽10选3）→ 穿越特效 → 身份生成 → 属性初始化
```

属性初始化规则：
- 声望：根据出身阶层（贵族 500 / 平民 100 / 贱籍 50）
- 财富：根据出身阶层 × 城市经济系数
- 学识：根据识字率（城市识字率 × 100）
- 颜值：随机 3000-7000
- 专属属性：全部 0（从零开始积累）

### 5.2 核心循环

```
AI叙事（含当前朝代真实事件）
  ↓
给出选项（2-4个）+ 自由输入
  ↓
玩家选择
  ↓
AI生成结果 + 属性变化
  ↓
显示属性增长 + 榜单排名变化
  ↓
检查死神危险度 → 可能触发死神事件
  ↓
下一回合
```

### 5.3 死亡与结算

```
玩家死亡（寿终/意外/被杀）
  ↓
本世总结：
  - 活了X岁
  - 获得属性总值
  - 上榜X个，最高排名X
  - 获得历史庇护X层
  ↓
墓志铭生成
  ↓
进入下一世（保留历史庇护层数）
```
