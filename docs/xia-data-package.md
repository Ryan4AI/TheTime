## 4. era_age_dist（年龄分布）

基于二里头可鉴定年龄38例人骨：
- 未成年人(<15): 18/38 = 47%
- 壮年(15-49): ~19/38 = 50%
- 老年(50+): 1/38 = 3%
- 平均年龄: ~23岁

每条 age 的 weight 由上述比例经线性插值得出。
（0-14岁各年龄从47%均摊；15-49从50%均摊；50+从3%均摊）

```json
[
  { "year": -1750, "age": 0, "weight": 3.1, "source": "据袁靖(2023), 76例二里头人骨年龄鉴定数据线性插值推算" },
  { "year": -1750, "age": 1, "weight": 3.1, "source": "同上" },
  ...（0-14岁各3.1, 共15条）
  { "year": -1750, "age": 15, "weight": 1.4, "source": "同上" },
  ...（15-49岁各1.4, 共35条）
  { "year": -1750, "age": 50, "weight": 0.6, "source": "同上" },
  ...（50-80各0.6, 共31条）
]
```

合计 81 条（0-80岁），总权重接近 100。
